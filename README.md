# 2D Array Multiplication Table Generator

A simple, clean, and efficient C++ console application that generates and prints a formatted multiplication table from 1 to 10 using a two-dimensional array.

## 🚀 Features
* **2D Array Storage:** Demonstrates how to pass and manipulate 2D arrays across functions.
* **Smart Indexing:** Uses custom math logic `(i + 1) * (j + 1)` to avoid multipling by zero while maintaining standard 0-based array limits.
* **Clean Formatting:** Uses formatted `printf` output to ensure all columns align perfectly.

## 💻 Code Structure
* `MultiplicationTable()`: Fills the matrix with multiplication data.
* `PrintArray()`: Handles the clean, tabulated console output.
* `main()`: Coordinates the workflow and handles memory allocation.

## 🛠️ How to Run
1. Open `Multiplication-Table.slnx` or `Multiplication-Table.vcxproj` in Visual Studio.
2. Build and run the project (Ctrl + F5).
